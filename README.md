# X Auto Like & Retweet

X (Twitter) üzerinde belirlediğiniz hesapların tweetlerini otomatik olarak beğenen ve retweet eden Chrome eklentisi.

---

## Kurulum

1. [**Eklentiyi indir**](https://github.com/ahmetozmtn/x-like-extionsion/releases/download/v0.1/x-like-extension-v0.1.zip)
2. ZIP dosyasını bir klasöre çıkart
3. Chrome'da `chrome://extensions` adresine git
4. Sağ üstten **Geliştirici modu**'nu aç
5. **Paketlenmemiş öğe yükle**'ye tıkla
6. Çıkarttığın `x-like-extension` klasörünü seç

---

## Kullanım

### Hesap Ekleme

**Yöntem 1: Popup'tan**

1. Tarayıcıdaki eklenti ikonuna tıklayın
2. Kullanıcı adını yazın (örn: `username`)
3. **Ekle** butonuna tıklayın

**Yöntem 2: Tweet üzerinden**

1. X ana sayfasında herhangi bir tweet'e gidin
2. Kullanıcı adının yanındaki **(+)** butonuna tıklayın
3. Hesap otomatik olarak listeye eklenir

### Ayarlar

| Ayar             | Açıklama                             |
| ---------------- | ------------------------------------ |
| Otomatik Beğen   | Tweetleri otomatik beğenir           |
| Otomatik Retweet | Tweetleri otomatik retweet eder      |
| Otomatik Mod     | Sayfa kaydırılırken otomatik çalışır |

### Manuel Tarama

**Şimdi Tara** butonuna tıklayarak sayfadaki tweetleri anında tarayabilirsiniz.

---

## Notlar

-   Eklenti sadece `x.com` ve `twitter.com` adreslerinde çalışır
-   Spam algılanmaması için işlemler arasında otomatik bekleme süresi vardır
-   Daha önce işlenen tweetler tekrar işlenmez
-   Eklenen hesaplar popup ve sayfa arasında senkronize olur
